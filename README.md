# PPCM App

Appli de décomposition en facteurs premiers et de recherche de nombres premiers dans une plage.

<br />

J'ai remis à jour les version les plus récentes de React, Node et Tailwind. Prise en compte de la
vulnérabilité CVE-2025-66478 (NextJS) et CVE-2025-55182 (React).

J'ai également ajouté MUI à l'ensemble.

<br>

## <u>Important Notes</u>

- Be sure to update the `package.json` to include your own information for the project you're working on.
- Purge CSS will only run in production, so use either `build` or `stage` to run it and minimize your CSS file size. You can also modify the `postcss.config.js` file to enable Purge CSS in development.
